# 🚀 Mini Test Platform

一个基于 FastAPI + Pytest + Allure 的自动化测试平台。

## ✨ 功能

- 自动化接口测试
- Allure 测试报告
- 中英文双版本报告
- GitHub Actions CI/CD
- GitHub Pages 在线展示

## 📊 在线报告

👉 https://E3N4U.github.io/mini-test-platform/

## 🛠 技术栈

- FastAPI
- Pytest
- Allure
- GitHub Actions

## 🚀 本地运行

```bash
pytest --alluredir=allure-results
allure serve allure-results
